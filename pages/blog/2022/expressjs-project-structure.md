Express is a Node.js web framework that provides a set of robust features for backend applications.

This is the structure I tend follow for any express application.

```sh
src/
   ├── controllers
   ├── index.ts
   ├── middlewares
   ├── routes
   ├── schema
   ├── __tests__
   └── utils
```
